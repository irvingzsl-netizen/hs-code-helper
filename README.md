# HS编码小助手 🔍

找编码，so easy！一个搞笑诙谐的现代极简风HS海关编码查询工具。

## ✨ 核心功能

- 🎨 **现代极简风** - 简洁清爽，不装X
- 🔍 **智能搜索** - 输入关键词秒找到
- 🏷️ **智能分类** - 不再把"塑料/橡胶"傻傻分不清，该是塑料就是塑料
- 🖱️ **拖拽排序** - 想怎么排就怎么排
- ✏️ **即点即改** - 点哪改哪，包括表头！
- ➕ **自由增删列** - 列不够？加！列太多？删！
- 💾 **自动保存** - 改完就存，不用担心
- 🤪 **搞笑文案** - 严肃？不存在的

## 🚀 使用方法

### 在线使用
直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何软件或服务器。

### 部署到GitHub Pages
1. 将代码上传到GitHub仓库
2. 进入仓库设置 Settings → Pages
3. 选择分支（通常是main）并保存
4. 等待几分钟后即可通过提供的URL访问

## 📱 怎么玩

### 搜索
- 在搜索框输入关键词就行
- 中英文、编码、啥都能搜

### 筛选
- 点类别标签筛选
- 可以多选
- 点"全部"清空

### 拖拽排序 ⭐
- 鼠标按住左边的 ⋮⋮ 拖动
- 想放哪放哪
- 松手自动保存

### 编辑单元格 ⭐
- 点任意格子就能改
- 回车或点别处保存
- ESC取消

### 编辑表头 ⭐⭐
- 点表头的列名也能改
- 改完所有数据都跟着变
- 悬停显示删除按钮（❌）

### 增删列 ⭐⭐
- 点"➕ 加一列"按钮添加新列
- 悬停表头点❌删除列
- 列的顺序就是显示顺序

### 增删行
- 点"➕ 加一条"添加新行
- 点行末的🗑️删除

## 💾 数据存哪了

- 存在浏览器的 localStorage 里
- 清浏览器数据=清这里的数据
- 换电脑=重新开始
- 建议定期截个图备份

## 📝 默认字段

- **类别**：塑料、橡胶、纸品、印刷、文具等（智能识别，不再混淆）
- **中英品名**：商品名称
- **材质/用途判断**：详细描述
- **HS6**：6位HS编码
- **10位申报**：完整申报编码
- **B3 Duty / MFN**：税率
- **口径**：归类口径
