# HS编码查询系统 🔍

一个支持**多人实时协同编辑**的现代极简风 HS 海关编码查询工具。

## 🌐 在线访问

**访问地址**：`https://你的用户名.github.io/仓库名/`

**访问密码**：`1140`

---

## ✨ 核心功能

- 🔐 **密码保护** - 只有授权人员可以访问
- 👥 **多人协同** - 基于 Firebase 实时同步，多人同时编辑
- 🔄 **实时同步** - 所有修改立即同步给所有在线用户
- 🎨 **现代极简风** - 简洁清爽的界面设计
- 🔍 **智能搜索** - 输入关键词快速查找
- 🏷️ **智能分类** - 按类别筛选查看
- 🖱️ **拖拽排序** - 自由调整数据顺序
- ✏️ **即点即改** - 点击任意单元格直接编辑
- ➕ **自由增删列** - 根据需要添加或删除列
- 💾 **自动保存** - 所有修改自动保存到云端

## 🚀 使用方法

### 在线使用（推荐）
1. 访问 GitHub Pages 部署的网址
2. 输入密码：`1140`
3. 开始使用！

### 本地使用
1. 克隆或下载此仓库
2. 双击 `index.html` 打开（本地模式，无多人协同）

## 📱 功能说明

### 🔍 搜索和筛选
- 在搜索框输入关键词进行全文搜索
- 点击类别标签进行筛选
- 支持组合筛选

### ✏️ 编辑功能
- **编辑单元格**：点击任意单元格即可编辑
- **编辑表头**：点击表头的 ✏️ 图标可以修改列名
- **添加条目**：点击"添加条目"按钮
- **添加列**：点击"添加列"按钮
- **删除行**：点击行末的 🗑️ 图标

### 🖱️ 拖拽排序
- 鼠标按住行左侧的 ⋮⋮ 图标
- 拖动到目标位置
- 松手自动保存

### 👥 多人协同
- ✅ 页面顶部显示连接状态
- ✅ 所有修改实时同步给所有在线用户
- ✅ 无需刷新页面即可看到其他人的修改
- ✅ 数据保存在云端，永不丢失

---

## 💾 数据存储

- **云端存储**：数据存储在 Firebase Realtime Database
- **本地缓存**：同时保存在浏览器本地作为缓存
- **离线支持**：网络断开时可以查看本地缓存
- **自动同步**：网络恢复后自动同步数据

## 📝 部署到 GitHub Pages

1. Fork 或上传代码到你的 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支
4. 点击 Save
5. 等待 1-2 分钟，访问生成的网址
